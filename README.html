Local setup

Run locally

Prerequisites

  • Python 3.12 + uv
  • GCP service account JSON with Vertex AI permissions (place at repo root, see .env.example)

Install & start

git clone <this repo>
cd agentic-awm-advisor

# install deps (uv resolves from uv.lock)
uv sync --extra dev

# copy and fill env
cp .env.example .env

# verify Vertex AI auth
uv run --no-sync python scripts/smoke_vertex.py

# start the server (FastAPI + React served from /static)
uv run --no-sync uvicorn app.main:app --port 8001 --reload
# open http://localhost:8001

Rebuild the brief & index

# re-run the 5-stage agentic flow and write brief cache
uv run --no-sync python -m app.orchestrator

# rebuild the Qdrant + BM25 index from corpus
uv run --no-sync python scripts/build_index.py

Tests

# 67 offline tests (no live Vertex calls)
uv run --no-sync pytest -m "not live" -q
